⛪ Landing Page - Assembleia de Deus Campinas Ponta Kayana

Projeto de uma landing page institucional desenvolvido para apresentar informações da igreja Assembleia de Deus Campinas Ponta Kayana, com foco em comunicação clara e acessível para membros e visitantes.

🚀 Funcionalidades

- Apresentação institucional da igreja
- Informações sobre cultos e horários
- Seção de contato
- Layout responsivo (adaptável para mobile)

🎯 Objetivo

Facilitar o acesso às informações da igreja, proporcionando uma presença digital simples, organizada e eficiente.

💡 Contexto

Projeto desenvolvido como landing page para divulgação e comunicação institucional, com foco em experiência do usuário e acessibilidade.

🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

🌐 Acesso ao projeto

https://assembleia-deus-campinas-pontakayana.vercel.app/

📌 Diferencial

Projeto focado em comunicação clara e navegação intuitiva, atendendo a necessidade de presença digital de uma instituição.
